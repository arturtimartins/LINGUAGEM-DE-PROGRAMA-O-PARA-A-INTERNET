# Gerenciador de Tarefas

Este é um projeto simples de Gerenciador de Tarefas desenvolvido em React.js.

## Requisitos

*   Node.js (versão 14 ou superior)
*   npm ou yarn

## Instalação

1.  Navegue até a pasta do projeto:
    \`\`\`bash
    cd gerenciador-de-tarefas
    \`\`\`

2.  Instale as dependências. Se estiver usando `yarn`:
    \`\`\`bash
    yarn
    \`\`\`
    Ou se estiver usando `npm`:
    \`\`\`bash
    npm install
    \`\`\`

## Execução

Para iniciar o servidor de desenvolvimento:

\`\`\`bash
yarn start
# ou
npm start
\`\`\`

O aplicativo será aberto automaticamente em seu navegador em [http://localhost:3000](http://localhost:3000).
